class qtmpl.MyMplCanvas(parent=None, width=5, height=4, dpi=100)¶

This creates a figurecanvas which has a canvas for plots and a navigation toolbar. These are widgets which can then be inserted into a PyQt application.

class qtmplex.plotview(parent=None, cmdpipes=None)¶

This class implements a simple matplotlib figure that updates every 500 milliseconds. The updating plot can be started, stopped and cleared in the interface to demonstrate possible use cases.
